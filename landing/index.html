🔒 KVKK + GDPR

Veri tarayıcınızda kalır

Bluedev'in sunucusu yok. Eklentinin manifest'inde tanımlı tüm host izinlerini chrome://extensions üzerinden inceleyebilirsiniz. AI istekleri tarayıcınızdan doğrudan seçtiğiniz sağlayıcıya gider — aracı sunucu kullanılmaz.

  • ✓ İlk açılışta KVKK rıza ekranı zorunlu
  • ✓ "Kayıtsız kişileri dahil et" varsayılan KAPALI
  • ✓ API anahtarları sadece chrome.storage.local
  • ✓ Hata logu max 50 kayıt, otomatik döner
Aydınlatma Metni → 
// manifest.json (alıntı)
"host_permissions": [
  "https://web.whatsapp.com/*",
  "https://api.anthropic.com/*",
  "https://api.openai.com/*",
  "https://generativelanguage.googleapis.com/*",
  "https://api.groq.com/*",
  "http://localhost/*"
]
Her host izni şeffaftır; AI sağlayıcısı seçmediğiniz sürece o domain'e istek yapılmaz.

Kurulum

Chrome Web Store yayını onay bekliyor. O zamana kadar dev mode kurulumu:

  1. GitHub release indir wa-contacts-exporter-1.0.0.zip dosyasını çıkar.
  2. Chrome'da uzantılar sayfası Adres çubuğuna chrome://extensions yaz, Enter.
  3. Geliştirici modu Sağ üstteki "Geliştirici modu" anahtarını aç.
  4. Paketlenmemiş yükle "Paketlenmemiş öğe yükle" tıkla → çıkardığın dist/ klasörünü seç.
  5. WhatsApp Web'i aç + KVKK rıza web.whatsapp.com adresine git, eklenti popup'ında rıza ekranını onayla.
Beta zip indir (yakında) CWS yayını sonrasında bu adım tek tıkla "Chrome'a Ekle" butonuna dönüşecek.

Sıkça sorulanlar

Bu Meta'nın resmi uzantısı mı?

Hayır. Bluedev tarafından geliştirilmiştir; WhatsApp / Meta ile bağlantılı değildir, sponsorluk almaz. Yalnızca sizin kendi WhatsApp Web oturumunuzdaki kendi verilerinize erişir. WhatsApp markası WhatsApp LLC'ye aittir.

Verilerim güvende mi?

Evet. Eklentinin manifest'inde web.whatsapp.com dışında yalnızca seçtiğiniz AI sağlayıcısının domain'i için istek yapılır. Bluedev'in sunucusu yok, aracı sunucu kullanılmaz, hiçbir telemetri toplanmaz.

Toplu pazarlama / spam için kullanılabilir mi?

Hayır — tasarımı buna karşı. Otomatik cevap modülü taslak modunda çalışır, rate limit + sessiz saat zorunludur, sohbet bazlı opt-in gerekir. KVKK uyum yükümlülüğü tamamen kullanıcıya aittir.

Yapay zeka için ücret ödemem gerekiyor mu?

Hayır. Yerel Ollama / LM Studio ücretsizdir. Bulut isteyenler için Gemini ve Groq'un ücretsiz tier'ları yeterli. Claude / ChatGPT için kendi API anahtarınızı kullanırsınız.

Mesajların tamamı export ediliyor mu?

WhatsApp Web yalnızca son ~50 mesajı RAM'de tutuyor. Daha eski geçmiş için sohbeti yukarı kaydırın veya "Eski mesaj yükleme turu = 5+" ayarını kullanın. Eklenti hafıza sınırını aşamaz; bu Meta'nın WA Web tarafındaki tasarım kararıdır.

Beta sürecinden sonra ne olacak?

Beta süresi boyunca ücretsiz kalacak. Geri bildirimleri toplayıp v0.2.0'da Pro tier seçenekleri sunmayı planlıyoruz (ekip özellikleri, gelişmiş AI). Beta kullanıcıları Pro'ya erken erişim hakkı kazanacak.
