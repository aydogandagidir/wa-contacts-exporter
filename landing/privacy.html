KVKK / GDPR Aydınlatma Metni

Son güncelleme: 6 Mayıs 2026 · v0.1.0

Önemli: Bu eklenti çıkarılan hiçbir veriyi sunucuya göndermez, üçüncü taraflarla paylaşmaz, bulut tabanlı bir hizmet kullanmaz (sizin açıkça seçtiğiniz AI sağlayıcısı hariç). Tüm işlem cihazınızda yereldir.

1. Veri sorumlusu

Bu eklentiyi geliştiren Bluedev, eklentinin sizin cihazınızda çalışan bir yazılım parçası olduğunu ve eklenti kullanımında oluşturulan veriler üzerinde hiçbir sunucu işleme gerçekleştirmediğini beyan eder. Eklenti sadece sizin etkileşim ettiğiniz WhatsApp Web sayfasındaki verilere erişir.

2. Hangi veriler işlenir?

Sizin tetiklediğiniz "Çıkar" işlemleri sırasında WhatsApp Web'in kendi Store nesnesinden okunan veriler:

  • Sohbet adları, telefon numaraları, son mesaj zaman damgaları
  • Grup adları, üye listesi, üyelerin telefon numaraları ve admin durumu
  • WhatsApp etiketleri ve hangi sohbete uygulandığı
  • Kayıtlı kişiler (Contact koleksiyonu)
  • Mesaj geçmişi (sohbet başına 100-5000, kullanıcı kontrollü)

Bu veriler yalnızca sizin tarayıcınızın chrome.storage.local alanında geçici olarak (popup oturumu boyunca) ve indirdiğiniz CSV/XLSX/VCard dosyasında saklanır.

3. AI sağlayıcı seçimi ve veri akışı

AI özelliklerini kullanırsanız, seçtiğiniz sağlayıcının domain'ine (Anthropic, OpenAI, Google, Groq veya kendi yerel Ollama / LM Studio sunucunuz) doğrudan tarayıcınızdan istek yapılır. Bluedev bu istekleri görmez ve aracılık etmez. API anahtarlarınız yalnızca yerel olarak chrome.storage.local'da saklanır.

4. KVKK / GDPR Sorumluluğu

Eklentiyi kullanarak çıkardığınız veriler, üçüncü kişilere ait kişisel verileri içerebilir (sohbet ettiğiniz kişilerin telefon numaraları, isimleri, grup üyeleri vb.). Bu veriler için:

  • İlgili kişilerden gerekli açık rızayı almak,
  • KVKK 5. madde veya GDPR 6. madde anlamında hukuki sebep belirlemek,
  • Verileri yalnızca işleme amacıyla sınırlı tutmak,
  • Verileri güvenli saklamak veya silmek

Tamamen kullanıcının (sizin) sorumluluğunuzdadır. Bluedev, kullanıcının yetkisiz veri işleme faaliyetlerinden hiçbir şekilde sorumlu değildir.

5. Varsayılan ayarlar

  • "Kayıtlı olmayan kişileri dahil et" anahtarı varsayılan kapalıdır.
  • Eklenti ilk açılışta KVKK rızası ister; rıza verilmeden hiçbir çıkarma yapılamaz.
  • Otomatik cevap modülü varsayılan kapalı; etkin olduğunda taslak modunda çalışır.
  • Hata logu yalnızca yerel chrome.storage.local'da, en fazla 50 kayıt; otomatik döner.

6. Verilerin silinmesi

Eklentinin tuttuğu yerel verileri silmek için chrome://extensions → "WA Contacts Exporter" → "Detaylar" → "Site verilerini sil" yolunu kullanabilirsiniz. İndirdiğiniz dosyalar bilgisayarınızda kalmaya devam eder; onları kendiniz silmelisiniz.

7. Meta / WhatsApp ile bağlantı

Bu eklenti resmi WhatsApp / Meta ürünü değildir ve WhatsApp LLC veya Meta Platforms Inc. tarafından desteklenmez, sponsorluk almaz veya bağlantılı değildir. WhatsApp markası WhatsApp LLC'ye aittir. Eklenti yalnızca kullanıcının kendi cihazındaki kendi WhatsApp Web oturumundaki kendi verilerine erişir.

Privacy Policy (GDPR / KVKK)

Last updated: 2026-05-06 · v0.1.0

Important: This extension does not send any extracted data to any server, does not share with third parties, and does not use any cloud-based service (except the AI provider you explicitly select). All processing is local on your device.

1. Data controller

Bluedev, the developer of this extension, declares that the extension is a piece of software running on your device, and that no server-side processing is performed on data generated through its use. The extension only accesses data on the WhatsApp Web page you interact with.

2. What data is processed?

Data read from WhatsApp Web's own Store object during user-initiated "Extract" operations:

  • Chat names, phone numbers, last-message timestamps
  • Group names, member lists, members' phone numbers and admin status
  • WhatsApp labels and which chat each is applied to
  • Saved contacts (Contact collection)
  • Message history (100-5000 per chat, user-controlled)

This data is stored only temporarily in your browser's chrome.storage.local (for the popup session) and in any CSV/XLSX/VCard file you download.

3. AI provider selection and data flow

When you use AI features, requests go directly from your browser to the domain of the provider you select (Anthropic, OpenAI, Google, Groq, or your own local Ollama / LM Studio server). Bluedev does not see or proxy these requests. API keys are stored only locally in chrome.storage.local.

4. GDPR / KVKK responsibility

Data extracted using the extension may include personal data of third parties (phone numbers, names, group members of people you chat with). For this data:

  • Obtain the necessary explicit consent,
  • Establish a lawful basis under GDPR Article 6 / KVKK Article 5,
  • Limit data use to the processing purpose,
  • Store securely or delete

Is entirely the user's (your) responsibility. Bluedev is not liable for any unauthorized data processing by the user.

5. Default settings

  • "Include unsaved contacts" defaults OFF.
  • The first run requires consent; without consent, no extraction is allowed.
  • Auto-reply module defaults OFF; when enabled, runs in draft mode.
  • Error log lives only in local chrome.storage.local, capped at 50 entries; rotates.

6. Data deletion

To delete the local data the extension keeps, go to chrome://extensions → "WA Contacts Exporter" → "Details" → "Clear site data". Files you've downloaded remain on your computer; delete them yourself.

7. Meta / WhatsApp affiliation

This extension is not an official WhatsApp / Meta product and is not affiliated with, endorsed by, or sponsored by WhatsApp LLC or Meta Platforms Inc. The WhatsApp trademark is owned by WhatsApp LLC. The extension only accesses data from the user's own WhatsApp Web session on the user's own device.
